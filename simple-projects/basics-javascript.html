What is javascript?

JavaScript, commonly called js, is a programming language that can add interactivity to a website. It was created by Brendan Eich (co-founder of the Mozilla project, the Mozilla Foundation, and the Mozilla Corporation).

But what is interactivity in a common website?

  • A game page
  • A page where you answer questions
  • A page that changes styles and/or have animations
  • Forms

So Javascript is what make pages interactive!
