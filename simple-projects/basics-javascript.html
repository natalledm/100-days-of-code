What is javascript?

JavaScript, commonly called js, is a programming language that can add interactivity to a website. It was created by Brendan Eich (co-founder of the Mozilla project, the Mozilla Foundation, and the Mozilla Corporation).

But what is interactivity in a common website?

  • A game page
  • A page where you answer questions
  • A page that changes styles and/or have animations
  • Forms

So Javascript is what make pages interactive!

This is just an experimental page. For detailed Javascript lessons, go to MDN.

Variables

Variables are containers that store values that can have different data types. You can declare a variable by calling var (not as common anymore) or let followed by the name you would give to the variable: let doAwesomeThing;

Other Items

We have other items as code:

  • Comments
  • Operators
  • Events

They are as important as Variables!

Conditionals

Conditionals are used to test if an expression returns true or not. A very common conditional is the if...else.

Functions

Functions are a way to save functionalities you want to repeat. This is a good alternative to repetition of the same code over and over.
